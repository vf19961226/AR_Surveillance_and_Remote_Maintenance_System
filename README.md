# AR_Surveillance_and_Remote_Maintenance_System_(Remote_Platform)
本分支儲存了遠端專家用於遠端協作的遠端平台，該平台以Unity遊戲引擎進行開發，並於其中以C＃ .NET腳本建立各項功能，以實現如遠端標註、語音通話、CNC姿態模擬、查看歷史數據等功能，以下將詳述如何安裝本程式以及遠端平台之基本操作。
